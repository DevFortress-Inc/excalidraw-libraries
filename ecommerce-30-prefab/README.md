# E-commerce 30 Prefab Library

A collection of 30 ready-to-use e-commerce design components for Excalidraw/Athena Board.

## What's Included

This library contains 30 professionally designed e-commerce screens in both Light and Dark themes:

### Light Theme (15 screens)
- Login
- Signup
- Home
- Catalog
- Product
- Cart
- Checkout
- Order Success
- Profile
- Settings
- Notifications
- Search
- Tracking
- Help Center
- Wishlist

### Dark Theme (15 screens)
All the same screens as above, optimized for dark mode.

## Installation

### Option 1: Direct Link
Click here to add to Excalidraw: [Add Library](https://excalidraw.com/?addLibrary=https://raw.githubusercontent.com/DevFortress-Inc/excalidraw-libraries/main/ecommerce-30-prefab/neo_code_ui_30_screens_v2_grouped.excalidrawlib)

### Option 2: Manual Installation
1. Download the `neo_code_ui_30_screens_v2_grouped.excalidrawlib` file
2. Open Excalidraw
3. Click on the library icon (books icon in the toolbar)
4. Click "Load library" and select the downloaded file
